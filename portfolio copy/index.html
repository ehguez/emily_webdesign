50/50 TYPE

50/50 is a typeface and its type specimen. Each character is mathematically constructed so that every letter is made up of the exact same amount of pixels. This reveals something about the structure of the alphabet, and allows me to code a specimen in such a way that the letters transition into pattern, and from one text, to another when the browser window is resized.
